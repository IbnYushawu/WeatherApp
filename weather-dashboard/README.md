**WEATHER DASHBOARD APP**
A responsive weather dashboard built with React, Vite, and plain CSS.
It allows users to search for any city and view real-time weather information using the OpenWeatherMap API.
Includes search history, dynamic weather backgrounds, and a mobile-responsive interface.

**LIVE DEMO**

https://capstoneweatherapp.netlify.app

**FEATURES**

- Search weather by city name

- View current temperature (°C)

- See humidity levels

- Check wind speed

- Weather condition icons

- Loading spinner while fetching data

- User-friendly error messages

- Responsive design (mobile, tablet, desktop)

- Search history (up to 5 cities)

- Delete individual cities from history

-Clear all search history

- Dynamic background images based on weather condition

**TECHNOLOGIES USED**

- React JS

- Vite

- Plain CSS

- JavaScript (ES6+)

- OpenWeatherMap API
